CAPITULO 1: INTRODUCCIÓN A LAS REDES DE COMUNICACIONES MÓVILES

Funciones Matemáticas de Señales

Función Rampa

La función rampa es una función continua que aumenta linealmente. Se define generalmente como \( r(t) = t \) para \( t \geq 0 \) y \( r(t) = 0 \) para \( t < 0 \).

Función Escalón

Es una función matemática que tiene valor 0 para \( t < 0 \) y valor 1 para \( t \geq 0 \). Se utiliza para representar señales que cambian abruptamente de valor.

Función Unitario

Es una función que tiene valor 1 para \( t \geq 0 \) y valor 0 para \( t < 0 \). Es una forma de representar una señal que se activa en \( t = 0 \).

Función Impulso

Es una función que tiene un valor infinito en \( t = 0 \) y 0 en todos los demás puntos. Se usa para modelar señales que ocurren instantáneamente.

Transformada de Fourier

Serie Trigonométrica

Representación de señales periódicas como una suma infinita de senos y cosenos. Es útil para analizar la frecuencia de señales.

Serie Exponencial

Se utiliza para representar señales en términos de exponenciales complejas. Es una forma alternativa a la serie trigonométrica.

Transformada de Fourier

La transformada de Fourier convierte una señal del dominio del tiempo al dominio de la frecuencia. Es fundamental en procesamiento de señales.
